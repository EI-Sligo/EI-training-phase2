Care of Instruments

Best Practices

  • Cleanliness: Wipe away oil, dirt, and sweat after use.
  • Storage: Keep in a dry environment (case) to prevent rust.
  • Protection: Protect sharp points and measuring edges. Don't stack them.
  • Lubrication: Light oiling of moving parts (e.g., micrometer spindle).

The Micrometer

  • Ratchet Stop: Always use the ratchet to apply consistent pressure. Never overtighten using the thimble.
  • Zero Check: Before measuring, close the anvil and spindle to check it reads 0.00.
  • Accuracy: Typically 0.01mm.
Reading: Sleeve (Main Scale) + Thimble (0.01 Scale).

Marking Out

Datum Line/Point

A fixed reference edge or point from which all measurements are taken to ensure accuracy.

Marking Blue

A dye applied to metal to make scribed lines visible. Safety: Use sparingly, avoid skin contact.

Key Tools

  • Scriber: Fine steel point for marking lines.
  • Centre Punch: Indents metal to seat a drill bit.
  • Protractor: For angular measurement.

The Hacksaw

Blade Orientation

Teeth must point FORWARD (away from handle). Cutting action is on the push stroke.

Technique

  • Use full length of blade.
  • Stance: Feet apart, balanced.

Filing

Types & Action

  • Single Cut: For finishing/sharpening.
  • Double Cut: For rapid material removal.
  • Cross Filing: Diagonal strokes for removal.
  • Draw Filing: Sideways motion for fine finish.

Safety & Care

  • Pinning: Clogged teeth scratch work. Clean with File Card.
  • Handle: Never use a file without a handle (Injury Risk).

Drilling Machines

Work Holding

Workpiece security is critical. Use a machine vice or clamps. Never hold work by hand.

Bit Holding

  • Jacobs Chuck: Straight shank drills. Remove key!
  • Morse Taper: Tapered shank drills. Use a Drift.

The Drilling Process

1. Centering

Always Centre Punch the mark to prevent drill wander.

2. Pilot Hole

For large holes (>10mm), drill a small pilot hole first to reduce pressure.

3. Coolant

Use soluble oil/coolant to reduce heat and preserve the bit.

4. Swarf

Use a brush to clear swarf. NEVER use bare hands.

Workshop Safety

Personal Safety

  • Safety glasses must be worn.
  • No gloves at rotating machines.
  • Correct stance for sawing/filing.

Workpiece Safety

  • Clamp work securely.
  • Use a brush for swarf (Razor sharp).
  • Remove chuck key immediately.

Piping Materials

Copper

Easy to bend. Low pressure air/water. Limitation: Reacts with Acetylene/Ammonia.

Stainless Steel

High tensile strength. High pressure/corrosive fluids. Work-hardens.

Cutting Pipe

Wheel Cutters create a clean cut but leave a large internal burr.

CRITICAL:

You MUST Ream (Deburr) the pipe after cutting. Burrs cause turbulence and damage O-rings.

Bending Pipe

Hand Bender

  • Match pipe size to former groove.
  • Calculate Setback for accuracy.
  • Bends: 45°, 90°, Offsets.

Precautions

  • Do not flatten/kink the tube.
  • Stainless Steel: Bend right the first time. Re-bending hardens it.
  • Watch for springback.

Fittings

Compression

Single Ferrule / Olive. Common in plumbing.

Twin Ferrule

Industry Standard (Swagelok/Parker).

  • Front Ferrule: Seals.
  • Back Ferrule: Grips.

Assembly Procedure

  1. Cut tube square and REAM.
  2. Insert into fitting until it bottoms out.
  3. Tighten nut Finger Tight.
  4. Mark the nut (6 o'clock).
  5. Tighten 1 and 1/4 turns (to 9 o'clock).

Danger: Never mix ferrule brands.

Inspection & Testing

Gap Gauge

Checks tightness (Go/No-Go).

Thread Tape

PTFE tape for threads. Apply clockwise.

Pressure Test

1.5x working pressure. Check leaks.
